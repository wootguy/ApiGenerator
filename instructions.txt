Header files may become invalid when the game updates. Follow these instructions to update them.

1) Download ASDocGenerator.exe from somewhere or compile it from the github source. Place  in this folder.
2) Run the game with the "-as_outputdocs" launch option to generate asdocs.txt
3) Run the python script. It will generate the angelscript docs and required plugin code.
4) Compile/download the ApiGenerator metamod plugin and install it
5) Add the ApiGenerator.as plugin to default_plugins.txt
6) Create the folder: svencoop/scripts/plugins/store/ApiGenerator
7) Start a map and type ".apigen"
8) Header files should be output to the the folder created in step 6
